# PyFy

PyFy is a lightweight desktop music player for Windows.  
It lets you play your local music files with a clean interface and simple controls.

No complicated setup â€” just download, install, and enjoy your music.

## Features

- Simple and modern interface  
- Play local audio files  
- Fast startup  
- Automatic installer  
- Desktop shortcut support  
- Runs directly from AppData (no system folders touched)
